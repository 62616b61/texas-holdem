# Texas Hold'em Poker Ranking

> Ranking of players in Texas Hold'em Poker

- [Features](#features)
- [How to use](#how-to-use)

## Features

- Identification of player's hand
- Ranking of players by hands
- Input validation
- Fancy output

## How to use

Sample input
```
KS AD 3H 7C TD
John 9H 7S
Becky JD QC
Sam AC KH
```

Run the app
```
npm start < input.dat
```

Output
```
1 Becky Straight Ace
2 Sam Two Pair Ace King
3 John Pair Seven
```
